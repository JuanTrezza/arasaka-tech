# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/2e4d9e79-f52e-4498-a172-3a9188f060dd

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## 🏢 Sobre el Proyecto

**ARASAKA TECH SOLUTIONS** es el sitio web corporativo B2B de la megacorporación más poderosa de Night City, año 2077.

Plataforma enterprise con soluciones de ciberseguridad neural, planes de servicio, casos de éxito y formulario de contacto corporativo. Diseñada para transmitir poder, tecnología y autoridad en el mercado B2B del futuro distópico.

> 💼 **Modelo de negocio:** B2B — empresa a empresa.

## ✨ Características

- 🔐 **Soluciones enterprise** de ciberseguridad neural para corporaciones
- 💎 **Planes de servicio** con pricing tiers diferenciados (Corporativo / Enterprise / Ejecutivo)
- 📊 **Casos de éxito** con métricas animadas y datos del mundo real
- 🖥️ **Formulario de contacto B2B** estilo terminal hacker
- 📈 **Stack tecnológico** con barras de progreso animadas
- 🏢 **Clientes corporativos** con scroll infinito y logos animados
- 🔢 **Stats con efecto countup** activado al hacer scroll
- 📱 **Menú hamburguesa responsive** para dispositivos móviles
- ⚡ **Efectos de glitch** en logo y overlay de scanlines cyberpunk

## 💼 Modelo de Negocio

**B2B (Business to Business)** — ARASAKA TECH SOLUTIONS vende soluciones tecnológicas y de ciberseguridad directamente a otras empresas y corporaciones.

No existen clientes individuales. Cada contrato se negocia a nivel ejecutivo, con acuerdos de confidencialidad y despliegue de infraestructura personalizada para cada corporación cliente.
